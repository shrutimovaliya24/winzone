## Image Folder Structure

Each page now uses its own image folder following this pattern:
- **Gallery images**: `/img/{page-name}-img/{page-name}-1.jpg`, `{page-name}-3.jpg`, `{page-name}-4.jpg`
- **Advantage image**: `/img/{page-name}-img/{page-name}-2.jpg`
- **Product card image**: `/img/{page-name}-img/{page-name}-1.jpg`
